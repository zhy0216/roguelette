# Buckshot Roulette（恶魔轮盘）— 原版规则

---

## 概述

Buckshot Roulette 是一款以霰弹枪俄罗斯轮盘为核心玩法的回合制策略游戏。玩家与荷官（Dealer）对坐，用一把装有实弹和空弹的霰弹枪轮流做出选择：射对手，或射自己。

---

## 基本规则

### 弹药

- 霰弹枪中装入若干 **实弹（Live）** 和 **空弹（Blank）**。
- 装弹时会**公开告知**实弹和空弹的数量，但弹药的**排列顺序是随机的**。

### 回合行动

每回合，当前行动者必须做出以下二选一：

| 行动 | 实弹结果 | 空弹结果 |
|------|---------|---------|
| **射对手** | 对手 -1 HP，回合结束，轮到对手 | 无事发生，回合结束，轮到对手 |
| **射自己** | 自己 -1 HP，回合结束，轮到对手 | 无事发生，**自己继续行动**（不交出回合） |

**核心机制**：射自己遇到空弹时保留行动权。这使"射自己"在空弹概率较高时成为有价值的策略选项。

### 胜负判定

- 某方 HP 归零 → 该方输掉本轮。
- 所有弹药耗尽且无人 HP 归零 → 重新装弹，继续对局。

---

## 游戏流程

### 单人故事模式

游戏分为 **3 轮（Round）**，每轮结构如下：

```
1. 血量分配阶段（Health Phase）
   分配本轮双方的 HP。

2. 道具分配阶段（Item Phase）
   双方各获得若干随机道具（第 1 轮无道具）。

3. 装弹阶段（Loading Phase）
   系统向霰弹枪装入实弹和空弹，公布数量。

4. 射击阶段（Shooting Phase）
   先手玩家开始行动：
   - 可以使用任意数量的道具（不消耗行动权）
   - 选择射击方向（对手 / 自己）
   - 根据结果决定行动权归属

5. 循环
   重复射击阶段直到某方 HP 归零或弹药耗尽。
   弹药耗尽 → 跳过血量分配，回到道具分配阶段，然后重新装弹。
   某方 HP 归零 → 该轮结束，存活方获胜。
```

### 三轮参数

| 轮次 | HP | 道具 | 说明 |
|------|-----|------|------|
| 第 1 轮 | 2 | 无 | 教学轮，纯概率对抗 |
| 第 2 轮 | 4 | 有 | 引入道具系统 |
| 第 3 轮 | 5 | 有 | 最终决战，道具更多 |

---

## 道具系统

### 通用规则

- 每次装弹前，双方各获得 **2-5 个**随机道具。
- 每人最多持有 **8 个**道具。
- 道具在行动前使用，**不消耗行动权**。
- 每回合可以使用**任意数量**的道具。
- 道具为一次性消耗品。

### 基础道具（所有模式）

| 道具 | 效果 |
|------|------|
| **放大镜（Magnifying Glass）** | 查看当前弹仓中弹药的类型（实弹或空弹） |
| **手锯（Hand Saw）** | 将当前实弹的伤害翻倍（仅对实弹生效） |
| **啤酒（Beer）** | 退出当前弹药（不射击），弹药类型会被揭示 |
| **香烟（Cigarettes）** | 回复 1 点 HP（不超过当前轮次的 HP 上限） |
| **手铐（Handcuffs）** | 锁住对手，跳过对手的下一个回合。已锁状态下不可再次使用 |

### Double or Nothing 模式追加道具

| 道具 | 效果 |
|------|------|
| **逆转器（Inverter）** | 将当前弹仓中的弹药类型反转（实弹↔空弹） |
| **一次性手机（Burner Phone）** | 揭示弹仓中某个**未来位置**的弹药类型（不能揭示当前弹药） |
| **过期药物（Expired Medicine）** | 50% 概率回复 2 HP，50% 概率扣 1 HP |
| **肾上腺素（Adrenaline）** | 偷取对手一个道具。需在限定时间内选择，超时则失败 |

### 多人模式追加道具

| 道具 | 效果 |
|------|------|
| **干扰器（Jammer）** | 阻止目标玩家在下一回合使用道具 |
| **遥控器（Remote）** | 强制目标玩家在下一回合射击指定方向 |

---

## 道具组合策略

### 经典组合

| 组合 | 操作 | 效果 |
|------|------|------|
| **放大镜 + 手锯** | 先看弹药，确认是实弹后挂手锯 | 确定性双倍伤害，消除不确定性 |
| **放大镜 + 射自己** | 先看弹药，确认是空弹后射自己 | 零风险保留行动权 |
| **啤酒 + 记弹** | 用啤酒退掉不利弹药，同时更新概率 | 操控弹仓内容 |
| **手铐 + 实弹** | 确认实弹后先用手铐再射对手 | 命中后对手无法立即反击 |
| **逆转器 + 概率判断** | 空弹概率高时使用逆转器，将大概率空弹变为实弹 | 概率操控 |

## 游戏模式

### 1. 故事模式（Story Mode）

- 与 AI 荷官（Dealer）对战。
- 3 轮制，难度递增。
- 第 1 轮无道具，第 2-3 轮有道具。

### 2. Double or Nothing（双倍或零）

- 通关故事模式后解锁。
- 无限模式：每次通关可以选择"带走奖金"或"继续挑战，奖金翻倍"。
- 追加 4 种新道具（逆转器、一次性手机、过期药物、肾上腺素）。
- 难度随轮次递增。

### 3. 多人模式（Multiplayer）

- 支持 2-4 人在线对战。
- 追加 2 种新道具（干扰器、遥控器）。
- 四人模式下每回合行动者可选择射击任意其他存活玩家或射自己。

## 核心策略要点

### 1. 记弹

记弹是最核心的技能。每次射击或使用啤酒退弹后，更新剩余的实弹和空弹数量，重新计算概率。

```
初始: L=3, B=5 → P(实弹) = 3/8 = 37.5%
退掉一个空弹后: L=3, B=4 → P(实弹) = 3/7 = 42.9%
又退掉一个实弹后: L=2, B=4 → P(实弹) = 2/6 = 33.3%
```

### 2. 射击方向的基本判断（无道具时）

- **P(实弹) > 50%** → 射对手（大概率命中）
- **P(实弹) < 50%** → 射自己（大概率空弹，保留行动权）
- **P(实弹) = 50%** → 射对手（更安全的默认选项）

### 3. 100% 确定时的行动

- **确定是实弹** → 射对手（必定命中）
- **确定是空弹** → 射自己（必定保留行动权）

### 4. 道具使用优先级

1. **手铐**：尽早使用，保证连续行动
2. **放大镜**：获取信息后再决策
3. **手锯**：确认实弹后使用
4. **啤酒**：在不利概率时用于退弹
5. **香烟**：低 HP 时使用
6. **过期药物**：通常避免使用（期望值为 +0.5 HP，但方差大）

## 数学基础

### 概率计算

```
P(实弹) = L / (L + B)
P(空弹) = B / (L + B)
```

其中 L = 剩余实弹数，B = 剩余空弹数。

### 简化模型下的最优策略

定义 W(L, B) = 当前行动者的胜率（单淘汰，无道具）：

```
W(L, B) = max(V_shoot_opponent, V_shoot_self)

V_shoot_opponent = L/(L+B) × 1 + B/(L+B) × (1 - W(L, B-1))
V_shoot_self     = L/(L+B) × 0 + B/(L+B) × W(L, B-1)
```

边界条件：W(L, 0) = 1

**定理**：在单淘汰无道具模型中，射对手的期望胜率永远不低于射自己。

### 胜率速查表

| L \ B | 0 | 1 | 2 | 3 | 4 | 5 |
|-------|---|---|---|---|---|---|
| **1** | 1 | 1/2 | 2/3 | 1/2 | 3/5 | 1/2 |
| **2** | 1 | 2/3 | 2/3 | 3/5 | 3/5 | 8/15 |
| **3** | 1 | 3/4 | 7/10 | 13/20 | — | — |
| **4** | 1 | 4/5 | 11/15 | — | — | — |

先手方胜率 W(L, B) ≥ 1/2 恒成立。
